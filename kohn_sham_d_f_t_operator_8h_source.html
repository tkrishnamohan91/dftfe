79 /**
80* @brief Compute discretized operator matrix times multi-vectors and add it to the existing dst vector

83 * interior nodes and exterior nodes to reduce memory access costs

99 * @param scalarA which is used for Chebyshev recursive iteration
100 * @param scalarB which is used for Chebyshev recursive iteration

131 * @param numberComponents number of wavefunctions associated with a given node
132 * @param processGrid two-dimensional processor grid corresponding to the parallel projHamPar
133 * @param projHamPar parallel ScaLAPACKMatrix which stores the computed projection
134 * of the operation into the given subspace
